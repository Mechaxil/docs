# Firmware (helyőrző)

Ez a projekt különválasztja a **helyi vezérlést** (Raspberry Pi Pico) és a **telemetriát** (ESP32-C3). A kódot a `firmware/` mappába kell tenni, és innen lehet majd hivatkozni rá.

## Pico (helyi vezérlés)
- Feladatai: szenzorok olvasása, terhelések vezérlése, alacsony fogyasztású alvó mód, RTC alapú időzítés, biztonsági lekapcsolás alacsony akkuszintnél.
- Javaslat: gyors prototípushoz MicroPython, élesebb teljesítményhez C SDK.
