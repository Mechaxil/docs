Raspberry Pi Pico W

SKU: MX-RP2W • Brand: Raspberry Pi • Type: Microcontroller Board

Highlights​

  • Dual-core RP2040 @ 133 MHz
  • 2.4 GHz Wi-Fi (Infineon CYW43439)
  • 26× multi-function GPIO
  • PIO state machines for custom protocols
  • Works with MicroPython, C/C++, and Rust

Getting Started​

# MicroPython blink example
from machine import Pin, sleep
led = Pin("LED", Pin.OUT)
while True:
    led.toggle()
    sleep(0.5)
