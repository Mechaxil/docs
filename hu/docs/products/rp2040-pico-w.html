Raspberry Pi Pico W

SKU: DEV-RPI-PICO-W • Brand: Raspberry Pi • Type: Microcontroller Board

Kiemelt jellemzők​

  • Kétmagos RP2040 @ 133 MHz
  • 2.4 GHz Wi-Fi (Infineon CYW43439)
  • 26 többfunkciós GPIO
  • PIO state machine-ek egyedi protokollokhoz
  • Működik MicroPython
    A Python egy kompakt változata mikrokontrollerekhez, például ESP32-höz vagy Raspberry Pi Picohoz. Közvetlenül a beágyazott hardveren lehet vele szkripteket futtatni.
    , C/C++ és Rust környezetben
